# Raffles Condominium Guest Guide

Welcome to your temporary home at Raffles Condominium. This repository contains the source code for our guest information portal, designed to make your stay seamless and enjoyable.

## Portal Features
- **Secure Access:** Password-protected guest experience.
- **Essential Information:** WiFi credentials, facility hours, and house rules.
- **Building Amenities:** Details on the infinity pool, gym, and common areas.
- **Quick Support:** Easy access to your host via WhatsApp.
- **Responsive Design:** Optimized for mobile and desktop viewing.

## Getting Started
The portal is a static single-page application. You can view it by hosting `index.html` on any static web server (e.g., GitHub Pages, Vercel, or Netlify).

## Contributing
While this is a private guest guide, we welcome suggestions for improvements. Please feel free to open a pull request if you have ideas to make the guest experience better!
